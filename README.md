# East Java Landcover QGIS

Peta penutupan lahan (landcover) Jawa Timur menggunakan QGIS.

## 📁 Struktur Folder

| Folder | Isi |
|--------|-----|
| `data/raw/` | Data mentah (shapefile, raster) |
| `data/processed/` | Data hasil olahan |
| `data/output/` | Output akhir (raster/shp/png) |
| `qgis-project/` | File project QGIS (.qgz) |
| `maps/` | Peta final (PNG, PDF) |
| `docs/` | Dokumentasi metodologi |
| `screenshots/` | Preview peta |

## 🛠️ Tools

- QGIS 3.x
- Data sumber: (isi sesuai sumber datamu)
